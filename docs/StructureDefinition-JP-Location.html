Resource Profile: JP_Location

Official URL: http://jpfhir.jp/fhir/core/StructureDefinition/JP_Location Version: 1.0.0
Active as of 2020-09-15 Computable Name: JP_Location

このプロファイルはLocationリソースに対して、所在場所のデータを送受信するための基礎となる制約と拡張を定めたものである。

Usage:

  • This Resource Profile is not used by any profiles in this Implementation Guide
