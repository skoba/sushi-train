Resource Profile: JP_MedicationDispense_Injection

Official URL: http://jpfhir.jp/fhir/core/StructureDefinition/JP_MedicationDispense_Injection Version: 1.0.0
Active as of 2021-12-25 Computable Name: JP_MedicationDispense_Injection

Copyright/Legal: FHIR® Japanese implementation research working group in Japan Association of Medical Informatics (JAMI)

このプロファイルはMedicationDispenseリソースに対して、注射薬剤処方調剤・払い出し記録のデータを送受信するための基礎となる制約と拡張を定めたものである。

このプロファイルはMedicationDispenseリソースに対して、注射薬剤処方調剤・払い出し記録のデータを送受信するための基礎となる制約と拡張を定めたものである。

Usage:

  • This Resource Profile is not used by any profiles in this Implementation Guide
