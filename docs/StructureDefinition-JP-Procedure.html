Resource Profile: JP_Procedure

Official URL: http://jpfhir.jp/fhir/core/StructureDefinition/JP_Procedure Version: 1.0.0
Active as of 2020-09-04 Realm: Computable Name: JP_Procedure

このプロファイルはProcedureリソースに対して、患者に対する医療的な処置のデータを送受信するための共通の制約と拡張を定めたものである。

Usage:

  • This Resource Profile is not used by any profiles in this Implementation Guide
