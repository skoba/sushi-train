Resource Profile: JP_Encounter

Official URL: http://jpfhir.jp/fhir/core/StructureDefinition/JP_Encounter Version: 1.0.0
Active as of 2021-10-25 Computable Name: JP_Encounter

このプロファイルはEncounterリソースに対して、来院/入院情報のデータを送受信するための基礎となる制約と拡張を定めたものである。

Usage:

  • This Resource Profile is not used by any profiles in this Implementation Guide
